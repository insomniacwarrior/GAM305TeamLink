Module Two Team Project Plan -
Scenario:
Top-Down

Theme:
Medieval Theme

Additional Elements:
•	Player Power-up Pickups 
•	Stationary Enemies 
•	Boss Enemy 
•	Obstacle (Traps)

Timeline:
Week two: 
Making a repository and gathering everyone to produce game themes and brainstorm.
Week three: 
Basic character movement with a block out of the shell of the level with simple props.
Week four: Implementing elements and polishing the blockout for the level.
Week five: 
More specialized mechanics branching off and implementing final elements, all core elements included and working.
Week six: 
Bug testing, polish, textures, cut scenes(optional), sound effects.

Alpha goal:
Having the player move throughout the level with simple interactions. Level blocked out with obstacles and enemies.
Beta goal:
Polishing the final game with all functions included in the game. The player should be able to play a fully textured game that includes the four additional elements that work properly.

Communication method: Discord server voice chat/text chat.
Method for assigning and reporting tasks: Spreadsheet and repository manager.
